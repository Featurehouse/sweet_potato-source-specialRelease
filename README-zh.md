## 版本
对于本模组不支持1.15.2及更低版本的问题我们十分抱歉，因为本模组为新版下界风格。<br />
本模组需要的软件配置有：
* Fabric api；
* Minecraft 1.16或1.16.1；
* Fabric loader（0.9.0+）。
